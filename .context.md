Node.js/Express web app for generating professional profile images (Freepik AI) + article carousels, integrated with Notion database of Rede Líderes leaders. Deployed serverless on Vercel.

## Summary

- `server.js` — Express dev server; routes `/api/generate`, `/api/proxy-image`, `/api/notion/search`
- `api/index.js` — Vercel serverless handler
- `app.html` — Main UI (profile image generator, carousel builder)
- `index.html` — Login/access screen
- `carrossel-artigo.html`, `minibio.html`, `mudanca-cargo.html` — Pre-built HTML templates
- `vercel.json` — Deployment config
- `fonts/` — Custom typography assets
- `package.json` — Dependencies (Express, etc.)

## Constraints

- MUST use `FREEPIK_KEY`, `FREEPIK_EDIT`, `NOTION_KEY`, `NOTION_DB` env vars
- MUST poll Freepik `/check` endpoint until task status is `completed`
- MUST proxy images via `/api/proxy-image` to bypass canvas CORS restrictions

## Guidance

- Dev: `npm start` → http://localhost:3000
- Notion search provides autocomplete by leader name
- Generated images are downloaded client-side from Freepik
