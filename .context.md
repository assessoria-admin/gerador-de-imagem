# gerador-de-imagem

## Propósito

Web app para geração de imagens de perfil profissional integrado à API da Freepik e ao banco de dados Notion da Rede Líderes.

## Stack

- **Runtime:** Node.js ≥ 18, Express 4
- **Deploy:** Vercel (serverless via `api/index.js`)
- **Dev local:** `npm start` → `http://localhost:3000`
- **APIs externas:**
  - Freepik (geração de imagem via IA) — env: `FREEPIK_KEY`, `FREEPIK_EDIT`
  - Notion (busca de pessoas) — env: `NOTION_KEY`, `NOTION_DB`

## Arquivos-chave

| Arquivo | Papel |
|---|---|
| `server.js` | Servidor Express local — rotas `/api/generate`, `/api/proxy-image`, `/api/notion/search` |
| `api/index.js` | Handler serverless para Vercel |
| `app.html` | Interface principal do gerador (frontend) |
| `index.html` | Tela de acesso/login |
| `vercel.json` | Configuração de deploy Vercel |

## Fluxo Principal

1. Usuário busca pessoa no Notion (autocomplete por nome)
2. App monta o prompt/payload para a Freepik API
3. Freepik retorna `taskId` → polling até status `completed`
4. Frontend faz download da imagem gerada
5. Proxy de imagem (`/api/proxy-image`) contorna CORS no canvas

## Variáveis de Ambiente

```
FREEPIK_KEY=...
FREEPIK_EDIT=...  (URL endpoint da API Freepik)
NOTION_KEY=...
NOTION_DB=...     (ID do database Notion com pessoas)
```
